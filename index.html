2. JavaScript and web tech at Skybox

Satellite Dashboards? What?

3:15-3:48

  • "Godfather screen": displays all the vital health telemetry from the satellites
  • Various charts to display telemetry readings over time, monitor trends
  • Sortable tables

  • initially engineers wanted to use Python, but the argument for using web tech was advanced. Whole MOC running with web tech.
  • started to get Node off the ground before acquisition

3. Challenges

A few tech challenges...

  • implementation: browsers choking on too much WebSocket throughput, browsers choking on trying to render too much data to sortable tables, etc (issue: DOM thrashing, view issues)
  • deployment: HUGE issue - JS in MOC was effectively the production version. SUPER locked down and separated environment. Physical airgap - all interaction had to be done inside physical MOC

People Problems

Most challenges not really related to tech stack choices. Biggest issues are people issues (dealing with their perception of JS), organizational issues, etc
